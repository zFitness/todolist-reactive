# TodoList-Reactive

使用 vite + pnpm + react + @formily/reactive 开发的 TodoList, 使用响应式开发，让你从 useState 中解放出来。

![](https://image.zmblog.wang/picbed/20220801144559.png)

## 快速开始
```bash
git clone https://github.com/zFitness/todolist-reactive
cd todolist-reactive
pnpm i
pnpm dev

```
